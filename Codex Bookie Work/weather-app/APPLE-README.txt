BOT DESTROYER Apple version

1. Install Node.js 18 or newer from https://nodejs.org/ if it is not already installed.
2. Unzip this folder on the Mac.
3. Double-click "Run BOT DESTROYER.command".
4. If macOS blocks it the first time, right-click the file, choose Open, then confirm.
5. The app opens at http://localhost:3003.

The Windows .bat launcher is still included for backup, but Mac users should use the .command launcher.
